NEMU KODE ANEH?
DECODE SEKARANG.

Jangan bingung melihat string acak. Terjemahkan Base64, Hex, Binary, atau URL Encoding menjadi bahasa manusia yang bisa dibaca dalam hitungan detik.

Protokol Dekripsi

  • 1
    Paste Ciphertext Tempel kode aneh yang kamu temukan (misal: "SGVsbG8=").
  • 2
    Select Mode Pilih metode dekripsi: Base64, Hex, Binary, atau Auto-Detect.
  • 3
    Decrypt Result Baca hasil terjemahan asli (Plaintext) di layar terminal.

Multi-Format

Mendukung Base64, Hexadecimal (Base16), Binary (0101), dan URL Decoding.

Dev Tools

Sangat berguna untuk men-debug API Token, JWT Header, atau parameter URL yang ter-encode.

100% Client-Side

Data sensitif atau kode rahasia kamu tidak pernah dikirim ke server. Proses dekripsi terjadi di browser.
